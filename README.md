### Installing

Clone the repository
```
git clone https://github.com/AMGrobelnik/MultiCometWeb
```
Navigate into the project's root directory:
```
cd MultiComet
```
Run the Github setup script (Downloads larger project files):
```
bash Setup/GithubSetup.sh
```
